# 🎮 Class Favorites Game

An interactive classroom voting game designed to help children practice basic English vocabulary in a fun and engaging way.

## 📌 About the Project

This project is a simple web-based game where students vote between two options (e.g., Pizza vs Hamburger).

It was created as a classroom activity to encourage:
- speaking
- vocabulary recognition
- participation

Students raise their hands and the teacher counts votes using the interface.

## 🎯 Features

- 🗳️ Two-choice voting system  
- ⏱️ Countdown timer (3-2-1 vote)  
- 🎉 Winner screen with animation and sound  
- 📊 Real-time vote counting  
- 🎨 Colorful and kid-friendly UI  
- 📱 Responsive design (works on mobile & tablet)

## 🧠 Educational Purpose

This game helps children:
- learn basic English words (food, school, daily life)
- practice speaking and expressing preferences
- stay engaged during lessons

## 🚀 How to Use

1. Open the `index.html` file in your browser  
2. Start the countdown  
3. Students vote by raising hands  
4. Count votes using + / - buttons  
5. Show the winner 🎉  
6. Move to the next question  

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript (Vanilla)

## 📸 Demo

You can run the project locally by simply opening the HTML file.
